Tasks API

https://github.com/luizdepra/go-rest-api

Operações

  • GET /tasks/ - Listar Tasks
  • POST /tasks/ - Criar Task
  • GET /tasks/:id/ - Obter Task
  • PUT /tasks/:id/ - Atualizar Task
  • DELETE /tasks/:id/ - Remover Task

Payload


{
  "id": 1,
  "title": "Make Coffee",
  "priority": 0,
  "done": false
}
